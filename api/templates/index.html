About

        

          This project demonstrates an end-to-end ML workflow: an RL policy trained on sequential market states,
          sentiment feature extraction from text, and a FastAPI service with a web UI for interactive inference.

FAQ

        

          
Why does it often recommend “Stay in cash”?

          

            The learned policy is intentionally conservative: cash has no downside and the agent invests only when
            state signals strongly favor it.
          


          
Is GitHub Pages able to run the API?

          

            GitHub Pages is static hosting only. The FastAPI server runs locally (or on a separate hosting provider).
